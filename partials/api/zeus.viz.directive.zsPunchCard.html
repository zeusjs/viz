zsPunchCard
directive in module zeus.viz

Description

A directive for a time based area polt

Usage

as element:
<zs-punch-card
       values="{Array.<number>}"
       [tick-formatter-y="{function($val,$pos)}"]
       [date-format="{string}"]>
</zs-punch-card>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
valuesArray.<number>

Values as array of numbers

tickFormatterY
(optional)
function($val,$pos

Callback function that accepts value and position of a data point and returns string

dateFormat
(optional)
string

Optional date format string
