zsBarGraph
directive in module zeus.viz

Description

A directive for horizontal bar charts

Usage

as element:
<zs-bar-graph
       values="{Array.<number>}"
       [tick-formatter-y="{function($val,$pos)}"]
       series-labels="{Array.<string>}"
       [left-margin="{number}"]
       [last-update="{Object}"]>
</zs-bar-graph>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
valuesArray.<number>

Values as array of numbers

tickFormatterY
(optional)
function($val,$pos

Callback function that accepts value and position of a data point and returns string

seriesLabelsArray.<string>

Y axis labels for each of the bars

leftMargin
(optional)
number

Optional left MARGINS for text. Defaults to 80

lastUpdate
(optional)
Object

Can be used to refresh the graph
