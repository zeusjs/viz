zsDonut
directive in module zeus.viz

Description

A directive for rendering single valued( percentage based ) donuts as well as multi series donut

Usage

as element:
<zs-donut
       percentage="{number}"
       values="{Array.<object>}"
       [segment-color="{function($val,$index)}"]
       [title="{string}"]
       [subtitle="{string}"]>
</zs-donut>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
percentagenumber

Percentage

valuesArray.<object>

Values as array of objects {label: 'tooltip text', value: 100}

segmentColor
(optional)
function($val,$index

Callback function that accepts value and index of a segment and returns its hex color as string

string
